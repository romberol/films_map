# Description
This module can be used to generate a map(html file) with the 10 movie locations nearest to the given coordinates.
## How to use
Use your command line run the script
```bash
python main.py 2000 49.83826 24.02324 locations.list
```
Where :
- 2014 - year of films (int)
- 49.83826 - latitude of your location (float)
- 24.02324 - longtitude of your location (float)
- locations.list - path to dataset (str)

## Result
A map, where each film is separate layer that can be hidden or shown.
